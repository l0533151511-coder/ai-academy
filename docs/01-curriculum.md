## טראק 0 — יסודות (Foundations)
*יעד: הבנה אמיתית של איך מחשב, אינטרנט וכלי פיתוח עובדים — בלי זה הכל שינון.*

**מודול 0.1 — יסודות מחשב**
1. מה זה בכלל מחשב — CPU/RAM/דיסק/ק"לט-פלט (אנימציה: מסלול הפעלת תוכנה)
2. מערכות הפעלה, תהליכים, זיכרון
3. איך קוד הופך לתוכנה רצה (קומפילציה מול אינטרפרטציה)
4. ייצוג מידע: בינארי, טקסט, תמונה, קול
5. **פרויקט:** דמיית "מחשב צעצוע" ויזואלית בדפדפן (הרצת פקודות בסיסיות על "זיכרון" מדומה)

**מודול 0.2 — איך האינטרנט עובד**
1. רשתות, IP, DNS (אנימציה: מסע חבילת מידע מהדפדפן לשרת ובחזרה)
2. HTTP/HTTPS, בקשה-תגובה
3. שרתים, קליינטים, דפדפנים
4. אבטחת תשתית בסיסית (TLS, תעודות)
5. **פרויקט:** מפה אינטראקטיבית שמראה "מסע בקשת HTTP" בזמן אמת

**מודול 0.3 — טרמינל ו-Linux**
1. שורת פקודה: ניווט, קבצים, הרשאות
2. פקודות ליבה (ls/cd/grep/find/pipes)
3. סקריפטים ואוטומציה בסיסית
4. סימולטור טרמינל אינטראקטיבי בתוך האתר
5. **פרויקט:** כתיבת סקריפט אוטומציה קטן שפותר בעיה אמיתית

**מודול 0.4 — Git ו-GitHub**
1. בקרת גרסאות — למה זה קיים (דיאגרמת ענפים אנימציה)
2. commit/branch/merge/rebase
3. Pull Requests, Code Review, קונפליקטים
4. GitHub Actions — טעימה ראשונה
5. **פרויקט:** תרומה מלאה ל"ריפו תרגול" כולל PR ו-merge
6. **🏆 Capstone #1:** בניית "תיק עבודות דיגיטלי" מוגרס בגיט — היסטוריית קומיטים נקייה, README מקצועי, דמו חי

## טראק 1 — יסודות ווב (Web Fundamentals)
**מודול 1.1 — HTML**: מבנה סמנטי, טפסים, נגישות, SEO בסיסי → פרויקט: עמוד נחיתה סמנטי מלא
**מודול 1.2 — CSS**: box model, flexbox, grid, responsive, אנימציות → פרויקט: בניית קלון UI מורכב מ-פיגמה
**מודול 1.3 — JavaScript**: משתנים/פונקציות/DOM/אירועים/async-await/fetch/closures/prototypes → פרויקט: אפליקציית To-Do מלאה עם localStorage
**מודול 1.4 — TypeScript**: טיפוסים, generics, interfaces, strictness, מיגרציה מ-JS → פרויקט: המרת פרויקט JS קיים ל-TS מלא
5. **🏆 Capstone #2:** אתר מסחר-אלקטרוני סטטי מלא (וניל HTML/CSS/TS) — רספונסיבי, נגיש, מהיר

## טראק 2 — הנדסת Frontend
**מודול 2.1 — React**: קומפוננטות, state/props, hooks, context, ביצועים → פרויקט: לוח מחוונים אינטראקטיבי
**מודול 2.2 — Next.js**: ראוטינג, SSR/SSG/ISR, Server Components, API Routes, אופטימיזציה → פרויקט: בלוג מלא עם CMS
**מודול 2.3 — ניהול State מתקדם ובדיקות Frontend**: Zustand/Redux, React Testing Library, E2E (Playwright) → פרויקט: אפליקציית ניהול משימות עם כיסוי בדיקות
5. **🏆 Capstone #3:** אפליקציית SaaS Frontend מלאה עם אימות, תשלומים מדומים, ולוח בקרה

---

## טראק 3 — הנדסת Backend
**מודול 3.1 — Node.js ו-REST APIs**: Express/Fastify, routing, middleware, error handling → פרויקט: API מלא ל-Blog
**מודול 3.2 — אימות והרשאות (Auth)**: JWT, OAuth2, sessions, RBAC → פרויקט: מערכת התחברות מאובטחת מלאה
**מודול 3.3 — מסדי נתונים ו-SQL**: מודל רלציוני, PostgreSQL, אינדקסים, טרנזקציות → פרויקט: סכמת DB מלאה למערכת CRM
**מודול 3.4 — MongoDB ו-Prisma**: NoSQL מול SQL, ORM, מיגרציות → פרויקט: מעבר API מ-raw SQL ל-Prisma
5. **🏆 Capstone #4:** Backend מלא ל-SaaS — REST API, DB, Auth, תיעוד OpenAPI

---

## טראק 4 — DevOps, ארכיטקטורה ואיכות
**מודול 4.1 — Docker ו-CI/CD**: קונטיינרים, Dockerfile, GitHub Actions pipeline → פרויקט: דוקריזציה מלאה + פייפליין אוטומטי
**מודול 4.2 — Deployment וענן**: Vercel/AWS/Cloud basics, סביבות, ENV vars, secrets → פרויקט: דיפלוי מלא לפרודקשן
**מודול 4.3 — ארכיטקטורת תוכנה ודפוסי עיצוב**: MVC, Clean Architecture, Design Patterns נפוצים, System Design בסיסי → פרויקט: תכנון ארכיטקטורה למערכת גדולה (מסמך + דיאגרמות)
**מודול 4.4 — בדיקות, דיבוג, ביצועים ואבטחה**: Unit/Integration/E2E, profiling, OWASP Top 10 → פרויקט: אבטחת מערכת קיימת + דוח בדיקות חדירה בסיסי
5. **🏆 Capstone #5:** לקיחת אחד הקאפסטונים הקודמים והפיכתו למוצר production-grade מלא: CI/CD, ניטור, אבטחה, דיפלוי אמיתי

## טראק 5 — יסודות AI (AI Foundations)
**מודול 5.1 — מבוא ל-AI ולמידת מכונה**: מהי בינה מלאכותית, סוגי למידה, מדדי הצלחה → פרויקט: מסווג פשוט בפייתון/JS
**מודול 5.2 — למידה עמוקה ורשתות נוירונים**: נוירון, שכבות, backprop, אימון (ויזואליזציה אנימטיבית של רשת לומדת) → פרויקט: זיהוי ספרות בכתב יד (בדפדפן, TensorFlow.js)
**מודול 5.3 — מודלי שפה גדולים (LLMs)**: טוקניזציה, טרנספורמר (דיאגרמת attention), context window, יכולות ומגבלות → פרויקט: כלי ניתוח טוקנים/עלויות אינטראקטיבי
5. **🏆 Capstone #6:** "מעבדת AI קטנה" — כלי שממחיש חיזוי, אימון וטוקניזציה בממשק אחד

---

## טראק 6 — הנדסת Prompt ופיתוח בעזרת AI
**מודול 6.1 — הנדסת Prompt מקצועית**: מבנה פרומפט, few-shot, chain-of-thought, הימנעות מהזיות → פרויקט: ספריית פרומפטים לשימוש חוזר עם A/B testing
**מודול 6.2 — Claude Code ופיתוח AI-assisted**: כלים, workflows, agentic coding, best practices → פרויקט: אוטומציה של תהליך פיתוח אמיתי בעזרת Claude Code
5. **🏆 Capstone #7:** בניית "Prompt Playground" מקצועי בתוך האקדמיה עצמה (self-referential project)

---

## טראק 7 — הנדסת אינטגרציית AI
**מודול 7.1 — MCP ו-Tool/Function Calling**: פרוטוקול MCP, schema של כלים, function calling patterns → פרויקט: שרת MCP מותאם אישית
**מודול 7.2 — Embeddings ומסדי נתונים וקטוריים**: מרחב וקטורי, similarity search, pgvector/Pinecone → פרויקט: מנוע חיפוש סמנטי
**מודול 7.3 — RAG (Retrieval-Augmented Generation)**: chunking, retrieval, ranking, הזיות מול דיוק → פרויקט: מערכת "שוחח עם המסמכים שלך"
**מודול 7.4 — מושגי Fine-tuning**: מתי צריך, LoRA, evaluation → פרויקט: הערכת מודל מותאם מול Prompt engineering
5. **🏆 Capstone #8:** מערכת RAG מלאה בפרודקשן — עם ניטור איכות תשובות

---

## טראק 8 — AI Agents ואוטומציה
**מודול 8.1 — סוכן AI בודד**: לולאת agent, זיכרון, תכנון, שימוש בכלים → פרויקט: סוכן משימות אישי
**מודול 8.2 — מערכות רב-סוכניות (Multi-Agent)**: תיאום, תקשורת בין סוכנים, orchestration → פרויקט: צוות סוכנים לפתרון משימה מורכבת
**מודול 8.3 — אוטומציה, Web Scraping ואינטגרציות API**: scraping אחראי, webhooks, שרשור שירותים → פרויקט: מערכת אוטומציה עסקית מקצה לקצה
5. **🏆 Capstone #9:** פלטפורמת אוטומציה מבוססת סוכנים למקרה שימוש עסקי אמיתי

---

## טראק 9 — מערכות AI בפרודקשן
**מודול 9.1 — ניטור, לוגים וסקייל**: observability, error tracking, load, caching → פרויקט: הוספת ניטור מלא למערכת קיימת
**מודול 9.2 — אופטימיזציית עלויות ואבטחת AI**: ניהול טוקנים/עלויות, Prompt Injection Defense, sandboxing → פרויקט: הקשחת מערכת AI קיימת + מבדק Red-Team עצמי
**מודול 9.3 — Best Practices לפרודקשן**: SLA, גרסאות מודל, feature flags, rollback → פרויקט: Runbook מלא + Playbook תקריות

## טראק 10 — פיתוח SaaS ומסחור (פרויקט גמר)
**מודול 10.1 — תכנון מוצר SaaS AI**: מיפוי משתמשים, תמחור, MVP scope
**מודול 10.2 — בנייה מלאה**: שילוב כל מה שנלמד — Frontend+Backend+DB+Auth+AI+Agents+RAG
**מודול 10.3 — עסקי**: billing, subscriptions, onboarding, analytics
**מודול 10.4 — השקה**: דיפלוי, ניטור, שיווק טכני, איסוף פידבק

**🏆🏆 פרויקט הגמר (Master Capstone):**
פלטפורמת AI SaaS מלאה ומוכנה-לפרודקשן — לבחירה מתוך: צ'אטבוט תמיכת לקוחות ארגוני / עוזר AI למסמכים / מערכת CRM חכמה / מערכת הזמנות עם סוכן AI. כוללת: Auth, DB, Backend API, Frontend מלא, אינטגרציית LLM, RAG, Agent, חיוב, ניטור, אבטחה, דיפלוי אמיתי לפרודקשן, תיעוד מלא ותיק שיווקי (README + Demo + Video pitch script).
