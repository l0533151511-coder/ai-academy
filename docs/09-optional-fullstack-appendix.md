# נספח אופציונלי — מסלול Full-Stack מלא (נדחה, לא בבנייה כרגע)

לפי [08-ai-focused-restructure.md](08-ai-focused-restructure.md), הוחלט למקד את האקדמיה בהנדסת AI ולא לבנות כברירת מחדל מסלול פול-סטאק מקצועי מלא. התוכן המקורי לטראקים 1-4 (יסודות ווב, הנדסת Frontend, הנדסת Backend, DevOps וארכיטקטורה — ~15 מודולים: HTML, CSS, JavaScript, TypeScript, React, Next.js, Node.js, Auth, SQL/PostgreSQL, MongoDB/Prisma, Docker/CI-CD, Deployment, System Design, Testing/Security) עדיין מתועד במלואו ב-[01-curriculum.md](01-curriculum.md).

**סטטוס:** לא נבנה. אם בעתיד המשתמש ירצה להעמיק במיומנויות פול-סטאק כלליות (מעבר למה שנחוץ ל-AI), נחזור למסמך זה כתוכנית המוצא ונבנה אותו מודול-אחר-מודול באותה מתודולוגיה.

**חלף אותו:** טראק 1' — "יסודות תכנות ל-AI" (מפורט ב-[08-ai-focused-restructure.md](08-ai-focused-restructure.md)), שמכסה רק את המינימום הנחוץ מ-JS/TS, Python, ו-REST APIs כדי לבנות ולהבין מערכות AI.
